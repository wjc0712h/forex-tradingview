## 이격도 정의

**이격도(Price Deviation Rate, Disparity Index)**는 현재 주가가 이동평균선으로부터 얼마나 떨어져 있는지를 백분율로 나타내는 기술적 분석 지표입니다.

### 핵심 개념
- 주가와 이동평균선 간의 상대적 거리 측정
- 과매수/과매도 상태 판단
- 평균 회귀 성향 활용
- 단기 추세 강도 측정

---

## 이격도 계산 방법

### 기본 공식
```
이격도 = (현재가 - 이동평균가) / 이동평균가 × 100
```

### 상세 계산 과정

#### 1. 단순 이격도
```
5일 이격도 = (오늘 종가 - 5일 이동평균) / 5일 이동평균 × 100
```

#### 2. 절댓값 이격도
```
절댓값 이격도 = |현재가 - 이동평균가| / 이동평균가 × 100
```

#### 3. 정규화 이격도
```
정규화 이격도 = (현재가 / 이동평균가 - 1) × 100
```

### 실제 계산 예시

**삼성전자 주식 예시**
- 현재가: 70,000원
- 20일 이동평균: 65,000원
- 이격도 = (70,000 - 65,000) / 65,000 × 100 = 7.69%

## 이격도 종류

### 1. 기간별 분류

#### 단기 이격도 (5일~20일)
- **5일 이격도**: 초단기 과열 상태 판단
- **10일 이격도**: 단기 추세 강도 측정
- **20일 이격도**: 월간 추세 대비 현재 위치

#### 중기 이격도 (30일~60일)
- **30일 이격도**: 중기 추세 강도 분석
- **60일 이격도**: 분기별 추세 대비 위치

#### 장기 이격도 (120일~240일)
- **120일 이격도**: 반기별 추세 분석
- **240일 이격도**: 연간 추세 대비 위치

### 2. 계산 방식별 분류

#### 단순 이격도
- 단순 이동평균 기준
- 가장 일반적인 형태

#### 지수 이격도
- 지수 이동평균 기준
- 최근 가격에 더 많은 가중치

#### 가중 이격도
- 가중 이동평균 기준
- 특정 기간에 더 많은 가중치

### 3. 시장별 분류

#### 개별 종목 이격도
- 개별 주식의 이격도
- 종목별 특성 반영

#### 지수 이격도
- 코스피, 코스닥 등 지수 이격도
- 시장 전체 과열도 판단

#### 섹터 이격도
- 업종별 이격도
- 섹터 로테이션 분석

## 이격도 해석

### 1. 기본 해석 원칙

#### 양수 이격도 (+)
- 현재가가 이동평균선 위에 위치
- 상승 추세 또는 과매수 상태
- 값이 클수록 과열 가능성 증가

#### 음수 이격도 (-)
- 현재가가 이동평균선 아래에 위치
- 하락 추세 또는 과매도 상태
- 값이 작을수록 침체 상태

### 2. 임계값 설정

#### 일반적인 임계값
- **+10% 이상**: 과매수 구간
- **+5% ~ +10%**: 상승 추세 구간
- **-5% ~ +5%**: 중립 구간
- **-10% ~ -5%**: 하락 추세 구간
- **-10% 이하**: 과매도 구간

#### 변동성 조정 임계값
```
고변동성 종목: ±15%
중변동성 종목: ±10%
저변동성 종목: ±5%
```

### 3. 시장별 해석 차이

#### 불시장 (Bull Market)
- 이격도가 높아도 추세 지속 가능
- 임계값을 상향 조정 필요

#### 베어시장 (Bear Market)
- 이격도가 낮아도 추세 지속 가능
- 임계값을 하향 조정 필요

## 이격도 활용 전략

### 1. 역추세 전략 (Contrarian Strategy)

#### 기본 원리
- 이격도가 극값에 도달하면 반대 방향 진입
- 평균 회귀 성향 활용

#### 매수 신호
```
조건: 이격도 < -10%
액션: 매수 진입
목표: 이격도 0% 회복
```

#### 매도 신호
```
조건: 이격도 > +10%
액션: 매도 진입 (공매도)
목표: 이격도 0% 회복
```

### 2. 추세 확인 전략

#### 상승 추세 확인
```
조건: 이격도 > +5% 지속
액션: 추세 추종 매수
목표: 추세 끝까지 보유
```

#### 하락 추세 확인
```
조건: 이격도 < -5% 지속
액션: 추세 추종 매도
목표: 추세 끝까지 보유
```

### 3. 다중 이격도 전략

#### 골든 크로스 변형
```
조건: 5일 이격도 > 20일 이격도
액션: 매수 신호
해석: 단기 모멘텀 강화
```

#### 데드 크로스 변형
```
조건: 5일 이격도 < 20일 이격도
액션: 매도 신호
해석: 단기 모멘텀 약화
```

### 4. 이격도 수렴/발산 전략

#### 수렴 패턴
- 이격도 변동폭이 줄어드는 상태
- 큰 움직임 전 신호
- 돌파 방향 확인 후 진입

#### 발산 패턴
- 이격도 변동폭이 커지는 상태
- 추세 강화 신호
- 추세 추종 전략 적용

## 이격도 한계점

### 1. 기술적 한계

#### 지연 지표 특성
- 이동평균 기반으로 후행성 존재
- 급격한 시장 변화 대응 어려움

#### 횡보장에서의 한계
- 추세가 없는 시장에서 신호 빈발
- 가짜 신호 (False Signal) 증가

### 2. 시장 환경별 한계

#### 강한 추세 시장
- 과매수/과매도 상태 장기간 지속
- 역추세 전략 손실 위험

#### 변동성 급증 시장
- 임계값 설정 어려움
- 급격한 이격도 변화로 대응 곤란

### 3. 계산상 한계

#### 이동평균 기간 설정
- 기간에 따른 민감도 차이
- 최적 기간 찾기 어려움

#### 임계값 설정
- 종목별, 시장별 차이 존재
- 고정값 사용의 한계

## 실전 활용 예시

### 1. 종목 선정 예시

#### A 종목 분석
```
현재가: 50,000원
20일 이동평균: 45,000원
이격도: +11.11%
판단: 과매수 구간, 조정 가능성
전략: 매도 또는 관망
```

#### B 종목 분석
```
현재가: 30,000원
20일 이동평균: 35,000원
이격도: -14.29%
판단: 과매도 구간, 반등 가능성
전략: 매수 검토
```

### 2. 포트폴리오 관리

#### 리밸런싱 기준
- 이격도 ±15% 도달 시 비중 조정
- 과매수 종목 비중 축소
- 과매도 종목 비중 확대

#### 위험 관리
- 이격도 극값 도달 시 포지션 크기 조정
- 손절매 기준으로 활용

### 3. 타이밍 전략

#### 진입 타이밍
```
1단계: 이격도 -10% 달성
2단계: 이격도 상승 전환 확인
3단계: 거래량 증가 확인
4단계: 매수 진입
```

#### 청산 타이밍
```
1단계: 이격도 +10% 달성
2단계: 이격도 하락 전환 확인
3단계: 거래량 감소 확인
4단계: 매도 진입
```

## 다른 지표와의 결합

### 1. RSI와 결합

#### 더블 과매도 신호
```
조건: 이격도 < -10% AND RSI < 30
신호: 강력한 매수 신호
확률: 반등 가능성 높음
```

#### 더블 과매수 신호
```
조건: 이격도 > +10% AND RSI > 70
신호: 강력한 매도 신호
확률: 조정 가능성 높음
```

### 2. 볼린저 밴드와 결합

#### 하단 밴드 + 이격도
```
조건: 볼린저 하단 밴드 터치 AND 이격도 < -8%
신호: 매수 신호
장점: 변동성 고려한 진입
```

#### 상단 밴드 + 이격도
```
조건: 볼린저 상단 밴드 터치 AND 이격도 > +8%
신호: 매도 신호
장점: 변동성 고려한 청산
```

### 3. 거래량과 결합

#### 거래량 급증 + 이격도
```
조건: 거래량 급증 AND 이격도 급변
신호: 추세 전환 가능성
전략: 추세 추종 또는 역추세
```

### 4. 이동평균과 결합

#### 다중 이동평균 + 이격도
```
5일 이격도: 단기 모멘텀
20일 이격도: 중기 추세
60일 이격도: 장기 추세
종합 판단: 다차원 분석
```

## 결론

이격도는 주가와 이동평균선 간의 상대적 거리를 측정하는 강력한 기술적 분석 도구입니다. 단순한 계산 방식에도 불구하고 다양한 투자 전략에 활용할 수 있으며, 다른 지표와 결합하면 더욱 효과적인 분석이 가능합니다.

### 핵심 요약
- **계산**: (현재가 - 이동평균) / 이동평균 × 100
- **해석**: ±10% 기준 과매수/과매도 판단
- **활용**: 역추세 전략, 추세 확인, 타이밍 전략
- **한계**: 지연성, 횡보장 한계, 강한 추세 시 주의
- **결합**: RSI, 볼린저 밴드, 거래량 등과 함께 사용

투자 시 이격도만 단독으로 사용하기보다는 다른 지표와 함께 종합적으로 판단하는 것이 중요하며, 각 시장과 종목의 특성을 고려하여 임계값을 조정하여 사용하시기 바랍니다.
